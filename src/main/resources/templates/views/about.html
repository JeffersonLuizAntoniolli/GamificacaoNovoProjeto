Dicas e Informações

Nesta Página estaremos lhe apresentando algumas informações que possam lhe auxiliar a utilizar a mexer no sistema. Vamos lá

Alguns Detalhes de Funcionamento do Protótipo de um simples Gerenciador de Atividades com Gamificação:

  • Não pode remover Projetos com Atividades Cadastradas nele;
  • Colaborador pode somente editar e remover suas próprias atividades que criou;
  • Quando Colaborador finalizar uma atividade que ele não tem afinidade, essa afinidade será atribuida para ele;
  • Na Conclusão de uma atividade, será atribuido '+1' para sua experiência de usuário e '+1' para sua experiência de afinidade atrelada à atividade;
  • Quando a atividade é desmarcada como completada, o usuário responsável perde '-1' da sua experiência de usuário e '-1' da sua experiência de afinidade;
  • Por padrão, todo novo usuário começa com '0' de experiência de usuário e das afinidades cadastradas;
  • Na Conclusão de uma atividade, será atribuido '+1' para o total de pontos que o usuário possui,;
  • Quando a atividade é desmarcada como completada, o usuário responsável perde '-1' dos seus pontos (Caso tiver 0, ficará negativo);
  • Por padrão todo usuário novo começa com 0 pontos;
  • Para comprar itens na loja, é necessário que o usuário tenha o numero de pontos igual ou superior ao custo do item que é apresentando;
  • O usuário poderá comprar mais de um mesmo item na loja;
  • Na página de perfil do usuário logado, é possivel visualizar dados pessoais e também a lista de: suas afinidades, seus itens comprados e suas atividades como reponsavel;
  • Para utilizar um item, acesse o perfil do usuário logado onde apresenta o 'email' no cabeçalho da página;
  • Um item uma vez que marcado como utilizado, terá sua situação alterada e ficará como registro no Historico do perfil de Usuário Logado;
  • Para atualizar uma imagem de avatar, acesse o perfil do usuário logado onde apresenta o 'email' no cabeçalho da página;
  • Não é recomendado querer tentar remover os usuários: ADMIN, MANAGER e GAMEMASTER;
  • Todos os usuários podem acessar o menu de 'Ranking' para visualizar os melhores colocados.

Missões para iniciar e aprender à mexer com o sistema:

  • 1º Missão: Acessar como usuário Administrador, vá no projeto "Aprendendo a mexer no sistema como Administrador do Sistema" e conclua todas as atividades;
  • 2º Missão: Acessar como usuário GameMaster, vá no projeto "Aprendendo a mexer no sistema como Game Master" e conclua todas as atividades;
  • 3º Missão: Acessar como usuário Colaborador, vá no projeto "Aprendendo a mexer no sistema como Colaborador" e conclua todas as atividades;
  • 4º Missão: Se divirta como quiser!! xD

Como Usuário Colaborador, você vai poder:

  • Comprar Itens da Loja;
  • Alterar imagem de perfil (Seu Avatar);
  • Ao completar atividades, adquirir pontos de compra e experiência de Afinidade;
  • Ao completar atividades, aumentar sua experiência total;
  • Vizualizar o Menu de Ranking;
  • Poder adicionar atividades para sí mesmo como Responsável.

Como Usuário Administrador ou Gerente de Projetos, você vai poder:

  • Mesmas ações que o Colaborador;
  • Cadastrar, Editar e Remover Projetos;
  • Cadastrar, Editar e Remover Atividades;
  • Encaminhar/Retirar Atividades para demais usuários;
  • Cadastrar, Editar (Dados Básicos) e Remover Usuário;
  • Concluir todas Atividades e todos Projetos.

Como Usuário Game Master, você vai poder:

  • Mesmas ações que o Colaborador;
  • Adicionar, Editar e Remover Afinidades;
  • Adicionar, Editar e Remover Itens da Loja.
